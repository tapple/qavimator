Menu Items - What do they do?

File

  • New
    Generates a new animation, resets all body parts to their default positions
  • Open
    Opens an existing .avm or .bvh animation
  • Add
    Adds another animation to the current scene, useful for animations that are meant to be used for more than one avatar
  • Save
    Saves the currently selected animation
  • Save As...
    Saves the currently selected animation under a new name
  • Load Props
    Loads a set of props into the current scene
  • Save Props
    Saves the props in the scene
  • Exit
    Exits the application

Edit

  • Undo (not yet working)
    Reverts the last editing operation
  • Redo (not yet working)
    Restores the last editing operation that has been undone before
  • Cut
    Cuts out the frame into the copy buffer
  • Copy
    Copies the current frame to the copy buffer
  • Paste
    Overwrites the current frame with the content of the copy buffer
  • Optimize BVH
    Tries to remove all unnecessary key frames from the animation to make it easier to modify again

Options

  • Skeleton
    Shows the internal avatar skeleton
  • Joint Limits
    Enables or disables the rotation limits for the joints. If enabled, you can only move a body part as far as it would go, if it was a real human body
  • Loop
    If this option is checked, the animation will start over from the beginning wile playing. If unchecked, the animation will stop at the end.
  • Protect First Frame
    If this option is checked the first animation frame will be protected (you see a red floor instead of the normal grey one). This is done to preserve the first frame as a "reference frame". Second Life in particular needs this reference to work properly. It will never be shown in the uploaded animation.
  • Show Timeline
    Show or hide the timeline with all keyframes and transitions
  • Configure QAvimator ...
    Opens the configuration dialog

Help

  • About
    Shows the version and build number of your copy of QAvimator
